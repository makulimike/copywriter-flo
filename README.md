# Copywriterflo - Copywriter Acquisition System

AI-powered client acquisition system for copywriters. Find leads, score them, send personalized emails, and book meetings automatically.

## Features

- 🤖 AI Lead Scoring (1-10 based on website analysis)
- 📧 Personalized Email Generation
- 🔍 Free Business Search (OpenStreetMap)
- 📊 Email Tracking & Open Detection
- 💬 Auto-Reply Detection & Analysis
- 📅 Meeting Scheduling
- 💰 One-time Payment ($280 Lifetime Access)

## Tech Stack

- Flask (Python)
- SQLite / PostgreSQL
- OpenAI API
- Stripe Payments
- OpenStreetMap API

## Local Development

1. Clone the repository
2. Create a virtual environment: `python -m venv venv`
3. Activate it: `source venv/bin/activate` (or `venv\Scripts\activate` on Windows)
4. Install dependencies: `pip install -r requirements.txt`
5. Create `.env` file with your keys
6. Run: `python app.py`
